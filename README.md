### 세부 역할
<details>
<summary>김건호(클라우드)</summary>
<div markdown="1">
- 서비스 기획<br>
- 클라우드 인프라 아키텍처 구축<br>
- Terraform을 이용한 IaC 방식으로 아래 인프라 구축(EC2 인스턴스, VPC, RDS 인스턴스)<br>
- Ansible을 이용한 인프라 노드 관리(EC2 인스턴스)<br>
- Github Actions를 통한 CI/CD 구현

</div>
</details>
<details>
<summary>박민엽(클라우드)</summary>
<div markdown="1">
- 서비스 기획<br>
- 전체 아키텍처 설계<br>
- AWS 네트워크 & 인프라 아키텍처 설계<br>
- Terraform을 이용한 인스턴스 구축<br>
- API Gateway, VPC Link로 내부 네트워크 연결망 구축

</div>
</details>
<details>
<summary>김은학(풀스택)</summary>
<div markdown="1">
- 서비스 기획<br>
- 이슈, PR 템플릿 작성<br>
- UI 와이어프레임 디자인<br>
- ERD 설계 및 데이터 활용 DB 구축<br>
- 추천 기능, 북마크 기능, 검색 기능 백엔드 API 구현<br>
- 카카오 맵 API, 구글 플레이스 API 활용 프론트엔드 설계

</div>
</details>
<details>
<summary>황윤미(풀스택)</summary>
<div markdown="1">
- 서비스 기획<br>
- 협업 툴 세팅<br>
- UI 와이어프레임 디자인<br>
- DB 및 API 설계<br>
- 회원관리(로그인, 로그아웃, 회원가입, 회원정보 수정)기능 백엔드 및 프론트엔드 구현

</div>
</details>
<details>
<summary>전우진(인공지능)</summary>
<div markdown="1">
- 서비스 기획<br>
- 프로젝트에 적합한 데이터 조사 및 수집<br>
- 데이터 전처리 작업 수행 (정규화, 데이터 증강 등)<br>
- 여행지 추천을 위한 최적의 모델 조사 (catboost regressor)<br>
- task에 맞춰 ML모델 세부 조정 및 tuning (weight 조정, 하이퍼파라미터 튜닝)<br>
- 모델 배포 및 api 서버 구현<br>
- 효과적인 코드와 알고리즘을 연구하고, 이를 기반으로 적절한 개선과 적용

</div>
</details>
<details>
<summary>박현혜(인공지능)</summary>
<div markdown="1">
- 서비스 기획<br>
- 프로젝트에 적합한 데이터 조사 및 수집<br>
- 데이터 전처리 작업 수행<br>
- 여행지 추천을 위한 최적의 모델 조사 및 Catboost 적용<br>
- 여행지 추천 모델 훈련 시 최적의 feature 선정<br>
- 기본 하이퍼파라미터로 모델을 학습하고 예측 결과를 확인<br>
- 훈련된 모델을 저장하고 Flask를 통해 REST API로 입출력 기능을 구현<br>
- 효과적인 코드와 알고리즘을 연구하고, 이를 기반으로 적절한 개선과 적용

</div>
</details>
